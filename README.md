# TFG Firmware
This is the repository for the firmware of the different boards in my TFG project.


## 📚 Required libraries
Here's a collection of the required libraries that are necessary to install in 
order to code and upload the firmware.

### Pico SDK
Provides libraries, headers, and CMake helpers to write and build firmware for
Raspberry Pi Pico boards.

### RISC-V GCC toolchain
Cross-compiler suite that translates C/C++ source into binaries executable on 
the Pico's RISC-V core.

### CMake
Creates the build files that are used by the ninja build tool. It's in charge
of figuring out compiler flags, source dependencies, include paths, and how to 
link libraries.

### Ninja
Compiles the source code into binaries using build instructions generated by 
CMake. That is, it compiles every .c file, links the resulting .o files into the 
final .elf and generates files in formats that are a derivative of .elf.

### picotool
Command-line utility for interacting with Raspberry Pi Pico boards, used to
upload firmware, reboot devices, inspect flash, and manage BOOTSEL mode.

### minicom
Serial terminal program to view logs and interact with the Pico over USB 
(UART/stdio).
